# linux ros command review

## 学习目标

    - 掌握linux操作和ros操作用于研究和开发软件包

### 文件操作

- `touch` make file
- `ls -l`list by condition
- `ls -F -R`list hidden file
- `cp -i`copy file
- `rm -rf directory` 删除文件夹及其所有内容
- `file my_file`check the file type
- `cat -n file`查看文件内容并显示行号
- `more file` 显示每页内容后停下

### bash shell

- `ps -p pidlist` 查看进程 `ps -A`
- `top`实时显示进程
- `kill PID`关闭进程
